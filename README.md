# Material-Anzeige

Webseite: https://stelomivo.github.io/MaterialAnzeige/

Tool zum Anzeigen und Filtern von Spielmaterial der Spielereihe [*Die Legenden von Andor*](https://legenden-von-andor.de/).\*

\*Diese Spielhilfe ist ein Fan-Projekt und steht in keinerlei Verbindung zu den Machern der Andor-Spielereihe.

## Bildquellen

Die Bilder stammen aus folgenden Quellen:
- Die Bilder im Ordner `Bilder/` stammen aus Begleitheften (https://fragkosmos.zendesk.com/hc/de/search?utf8=✓&query=ANDOR), Bonus-Material (https://legenden-von-andor.de/bonusmaterial/) u. Ä.
- Bei den Bildern im Ordner `Bilder/App` handelt es sich um Screenshots aus der Andor-App "Das Geheimnis des Königs" (https://andorgame.com/).
- Bei den Bildern im Ordner `Bilder/Spielplan` handelt es sich um Fotos aus dem Andor-Wiki (https://die-legenden-von-andor.fandom.com/de/wiki/Spielpläne).

Eine kommerzielle Nutzung der Bilder ist nicht gestattet.
